# KARIBU GROCERIES LTD (KGL)

A management system for wholesale cereal distribution.

## Features
- Login system
- Director dashboard
- User management
- Stock management
- Sales recording
- Credit sales
- Reports

## Technology Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Installation

1. Clone the repository
2. Install backend dependencies: `cd backend && npm install`
3. Create a `.env` file in the backend folder
4. Start MongoDB
5. Run the backend: `npm run dev`
6. Open `index.html` in your browser
